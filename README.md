# SimpleRender

OpenGL based render
For GLTF mesh loading a tinyGLTF library was used: https://github.com/syoyo/tinygltf

- Deffered shading, GBuffer
- 2 render passes: base and lighting
- BRDF specular
- Screen-Space occlusion
- Baked occlusion
- Exponential depth based fog
- Pseudo PBR
